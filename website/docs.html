本地编译部署三步曲

LepoProxy 的前端是一套极为严密和舒服的网页界面设计，后端是健壮的 Rust 提权驱动。您可以轻松按照下方命令在本地运行开发环境：

developer@lepoproxy:~ 
# 1. 克隆官方代码仓库
git clone https://github.com/WumiaoTech228/LepoProxy.git
cd LepoProxy

# 2. 极速安装前端 UI 构建依赖项
npm install

# 3. 启动开发服务器 (支持热重载与 Rust 自动重编译)
npm run dev

Windows 平台环境准备

  • Rustup (Rust 1.75+) 必须选择“C++ 桌面开发”工作负载以获得 MSVC 链接器。
  • Windows 11 / 10 SDK 编译 tauri 底层虚拟网卡网络接口所必须。
  • Node.js LTS (>=20.x) 网页前端开发热渲染引擎动力源。
